HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch names: creates a new branch and switches to it
  • ls: used to list files and directories
  • mkdir (file name): Used to create a new directory
  • cd: Used to navigate between directories
  • rm -r (file name): Used to remove files and directories
  • git init: Creates a brand new repo on your computer
  • git clone (url): Copies an existing repo from somewhere else to your local computer
  • git status: checks what branch we are currently on
  • git branch (branchname): creates a new branch
  • git checkout (branchname): used to change from one branch to another
  • git checkout -b (branch names): creates a new branch and switches to it
  • git commit: Commits files
  • git add -A: Commits all files in the working branch to GitHub
  • git commit -m"___": commits the repo with message
  • git pull origin (branch): pulls from the main repo and merges to the local branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
